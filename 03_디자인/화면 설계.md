## 페이지 구성

| 페이지       | 설명                               |
| --------- | -------------------------------- |
| 메인        | 검색바, 메타 덱 Top 8~10 |
| 덱         | 메타 덱 목록, 필터/정렬, 클릭 시 상세 펼침 (덱 구성, 아이템, 증강, 통계 등) |
| 전적 검색     | 소환사 검색, 프로필, 매치 히스토리, 매치별 상세     |
| 마이페이지    | 유저 프로필, 탭 구성(AI 리포트/전적), 결제, 리포트 목록 |
| AI 분석 상세 | 매치 분석 결과 — 종합 등급, 매치 컨텍스트, 운영/덱/아이템/증강 상세 분석 |

## 메인 페이지

### 레이아웃 (위에서 아래 순서)

**1. 상단 — 검색 + 패치 정보**
- 소환사 검색바 (메인 CTA, 중앙 배치)
- 현재 패치 버전 + 데이터 기준 정보 (예: "패치 14.21 · 52,341 매치 분석 · 30분 전 갱신")
- 티어 필터 (Platinum+ / Diamond+ / Master+ / Challenger) — 메인에서 선택하면 하위 섹션 모두 반영

**2. 메타 덱 티어리스트 Top 8~10**
- S/A 티어 덱만 표시
- 카드형: 덱 이름, 티어 뱃지, 주력 특성 아이콘, 캐리 챔피언 이미지, 순방률, 평균 등수
- 카드 클릭 → 덱 페이지에서 해당 덱 상세 펼침
- "전체 보기" 버튼 → 덱 페이지로 이동

### 갱신

- 통계 갱신(1시간)과 동일 주기
- 패치 전환 시 자동으로 새 패치 데이터로 전환 (이전 패치는 드롭다운으로 조회 가능)

### 후원

- 메타 덱 하단에 후원 박스 배치
- "롤체토스를 응원해주세요" + 설명 + 두 버튼 (커피 한 잔 후원하기 / Discord 참여)

## 덱 페이지

- 상단: 티어 필터 (Platinum+ 등) + 필터/정렬 옵션
- 덱 카드 목록: 티어 뱃지(S/A/B/C) + 덱 태그(Hot/꿀/New 등) 함께 표시
- 기본 정렬: 티어(복합 점수) 높은 순
- 필터: 티어 등급, 특성, 기물, 태그
- 정렬: 티어, 순방률, 평균 등수, 픽률, 1등 확률

### 덱 행 구성

- 순위 + 티어 뱃지 + 덱 이름 + 태그
- 챔피언 구성 (8~10칸, 대체 유닛 포함) + 아이템 표시
- 통계: 순방 확률, 평균 등수, 1등 확률, 선택률
- 팀 코드 복사 버튼 + 아코디언 펼침 아이콘

### 덱 상세 (아코디언 펼침)

- **추천 배치**: 헥스 보드에 챔피언 위치 표시
- **조합 아이템 우선순위**: 우선순위 순서로 나열, 필요 개수 표시
- **추천 증강**: 증강 이름 + 순방률
- **레벨별 가이드**: Lv.4~9 각 단계별 구성 가이드

## 전적 검색

### 모드 필터

- 상단 탭: 모든 게임 모드 / 일반 / 랭크 / 초고속 모드 / 더블 업
- 시즌 셀렉터 (시즌 14, 13, ...)

### 소환사 프로필

- 프로필 아이콘 + 소환사명#태그 + 랭크/LP + 상위 %/등수
- 플레이어 태그: 플레이 스타일 기반 뱃지 (hover 시 조건 툴팁)
  - 예: 후반 전문가, 치킨 수집가, 안전 운전 등

### 통계

- 핵심 지표 카드 4개: 1등 횟수, 순방 확률, 게임 수, 평균 등수 — 각 항목별 동일 티어 내 상위 % 표시
- LP 추이 그래프 (최근 N게임)
- 등수 분포 히트맵 + 막대 차트

### 시즌 기록 + 플레이어 분석

- 시즌별 최고 랭크, LP, 게임 수
- 플레이어 분석 레이더 차트 (1등 확률, 순방 확률, 골드 관리, 유연함, 평정심) — 각 축 S/A/B/C 등급 + 동일 티어 평균 비교

### 최근 통계

- 자주 사용한 덱 (최근 N게임)
- 자주 사용한 챔피언 (최근 N게임)

### 매치 히스토리

- 등수 (색상 구분: 1등 골드, 2~4등 청록 그라데이션, 5~7등 회색, 8등 빨강)
- 게임 모드, 경과 시간, LP 변동
- 전설이 아이콘 + 레벨, 매치 태그 (전설이 옆 배치)
- 시너지, 챔피언 구성 + 아이템
- 빠른 액션: 팀 코드 복사, 매치 링크 복사, AI 분석 버튼

### 매치 상세 (클릭 시 펼침)

- 매치 태그, 평균 티어
- 8명 참가자 정보: 등수, 전설이 + 레벨, 소환사명, 티어, 생존 시간 + 라운드, 피해량, 골드
- 시너지 + 챔피언 구성
- 참가자별 팀 코드 복사

### 매치 태그

매치별 성과 기반 태그 (한 판당 0~2개, 호버 시 조건 툴팁 표시)

| 태그 | 조건 | 색상 |
| --- | --- | --- |
| 도살자 | 플레이어 킬 5회 이상 | 빨강 |
| 삼위일체 | 3성 챔피언 3개 이상 | 골드 |
| 재앙 | 4코스트 챔피언 3성 달성 | 핑크 |
| 신 | 5코스트 챔피언 3성 달성 | 골드 |
| 부자 | 연속 이자 10골드 이상 유지 | 초록 |
| 과다출혈 | 4라운드 연속 체력 손실 | 적갈색 |
| 고밸류 | 평균 챔피언 코스트 3.5 이상 | 보라 |
| 덱 메이커 | 선택률 낮은 비주류 덱으로 순방 | 주황 |

## 마이페이지

### 유저 프로필

- 프로필 아이콘 + 소환사명 + 랭크/LP
- 전적 검색 프로필과 동일 구조

### 탭 구성

| 탭 | 내용 |
| --- | --- |
| AI 리포트 | 잔여 횟수 + 결제 + 리포트 목록 |
| 전적 | 통계, 차트, 매치 히스토리 (전적 검색과 동일 구조) |

### AI 리포트 탭

**결제:**
- 잔여 횟수 표시
- 회당 결제 (건당 구매)
- 구독 (프리즘 티켓, 월정액 무제한)
- 회원가입 시 무료 N회 제공

**리포트 목록:**
- 카드형 목록: 등수 + 챔피언 구성 + 아이템 + 종합/운영/덱/아이템/증강 등급 표시
- 클릭 시 AI 분석 상세 페이지로 이동
- 빠른 액션: 팀 코드 복사, 매치 링크 복사

## AI 분석 상세

- breadcrumb: 마이페이지 > AI 리포트 > 매치 분석

### 전적

- 해당 매치 카드 (전적 검색과 동일 구조, 클릭 시 8명 상세 펼침)

### 요약

- **종합 등급**: S/A/B/C + 한 줄 요약
- **카테고리별 등급 카드 4개**: 운영, 덱 선택, 아이템, 증강 — 각각 등급 + 짧은 설명

### 상세 분석

**매치 컨텍스트:**
- 판 전체 분위기 태그 (느린 템포, 고이자 로비 등)
- 로비 분석 (8명 평균 행동 패턴)
- 플레이어 판단 평가 (컨텍스트 대비 선택의 적절성)

**운영 분석 (클라이언트 수집 데이터 기반):**
- 등급 뱃지 + 분석 텍스트
- 라운드별 골드 변화 + 이자 유지율
- 레벨업 타이밍 (스테이지별 평가, 상위권 대비)
- 리롤 타이밍 + 효율 (골드 대비 획득 유닛 가치)

**덱 선택 분석:**
- 등급 뱃지 + 덱 티어 정보
- 로비 내 경쟁자 수 + 핵심 유닛 경합도
- 로비 메타 대비 선택 적절성

**아이템 분석 (클라이언트 + API 데이터):**
- 등급 뱃지 + 챔피언별 아이템 평가
- 아이템 조합 순서 + 완성 타이밍
- 현재 아이템 vs 추천 아이템 비교

**증강 분석:**
- 등급 뱃지 + 1차/2차/3차 증강 각각 평가
- 증강별 선택 근거 + 최적 여부 판정

## 공통 UX 패턴

### 로딩 상태

- **스켈레톤 UI** — 콘텐츠 영역의 형태를 유지한 채 placeholder 표시
- 페이지별 스켈레톤:
  - 메인: 검색바는 즉시 표시, 덱 카드 영역 스켈레톤
  - 덱 목록: 카드 스켈레톤
  - 전적 검색: 프로필 카드 + 매치 카드 스켈레톤
  - 매치 상세: 펼침 시 내부 스켈레톤

### 에러 상태

| 상황 | 메시지 | 액션 |
| --- | --- | --- |
| API 장애 / 네트워크 에러 | "문제가 발생했습니다" | 재시도 버튼 |
| 소환사 검색 실패 | "소환사를 찾을 수 없습니다" | 닉네임#태그 형식 안내 |
| Rate Limit 초과 | "요청이 많습니다. 잠시 후 다시 시도해주세요" | 자동 재시도 |
| 서버 점검 | "TFT 서버 점검 중입니다" | 자동 감지 |

### 빈 상태

| 상황 | 메시지 |
| --- | --- |
| 매치 히스토리 없음 | "최근 매치 기록이 없습니다" |
| 필터 결과 없음 | "조건에 맞는 덱이 없습니다. 필터를 조정해보세요" |
| 패치 초반 데이터 부족 | "새 패치 데이터를 수집 중입니다 (현재 N건)" |
| 꿀 덱 없음 | "현재 패치에서 발견된 꿀 덱이 아직 없습니다" |

### 반응형 웹

| 뷰포트 | 기준 | 레이아웃 |
| --- | --- | --- |
| 데스크톱 | 1024px 이상 | 전체 레이아웃, 사이드바 필터 |
| 태블릿 | 768~1023px | 2열 그리드, 필터 접기 |
| 모바일 | 767px 이하 | 1열 스택, 필터 모달, 터치 최적화 |

**모바일 주요 대응:**
- 덱 카드: 챔피언 아이콘 크기 축소, 가로 스크롤 없이 표시
- 매치 히스토리: 핵심 정보만 (등수, 덱 이름, 챔피언)
- 필터: 하단 모달 시트
- 매치 상세 8명 보드: 가로 스크롤 허용 또는 축소 뷰
- 툴팁: 호버 대신 탭
