> 판단 기준: **"상대방의 응답이 지금 당장 필요한가?"**
> - 필요하다 → REST (동기) — 전화처럼 바로 응답
> - 아니다 → Redis Streams (비동기) — 카톡처럼 나중에 확인

## 시나리오별 통신 흐름

**1. 유저가 메타 덱 페이지를 연다**

```
유저 → API Gateway ──REST──→ Comp Service ──→ Redis 캐시 → 덱 목록 반환
```

유저가 화면 앞에서 기다리고 있으므로 전부 REST.

**2. 유저가 소환사 전적을 검색한다**

```
유저 → API Gateway ──REST──→ Summoner Service ──REST──→ Riot API → 전적 반환
```

검색한 소환사가 DB에 없는 신규 유저라면:

```
Summoner ──Redis Streams──→ Collector ("이 소환사 매치 수집해줘")
```

유저에게는 Riot API로 바로 전적을 보여줌. 매치 수집은 백그라운드에서 나중에 처리하면 되므로 Redis Streams.

**3. 매치 데이터 수집 → 통계 집계 (배치)**

```
Collector가 매치를 계속 수집 중...
    ↓ 일정 건수 쌓임
Collector ──Redis Streams──→ Analytics ("매치 3,000건 쌓였어, 배치 돌려")
    ↓
Analytics ──REST──→ Collector ("그 매치 데이터 줘")
    ↓
Analytics가 클러스터링, 티어 산정 등 집계 수행
    ↓
Analytics ──REST──→ Comp ("새 덱 통계 여기 있어, 저장해")
    ↓
Analytics ──Redis Streams──→ Summoner ("통계 갱신됐어, 캐시 비워")
```

- 배치 트리거: Analytics가 바로 안 돌려도 됨 → Redis Streams
- 데이터 조회/전달: 배치 작업 중에 데이터가 필요 → REST
- 캐시 무효화: Summoner가 즉시 안 비워도 됨 → Redis Streams

**4. 유저가 AI 피드백을 요청한다 (유료)**

```
유저 → API Gateway ──REST──→ AI Feedback ──REST──→ Comp ("이 덱 통계 줘")
                                          ──REST──→ Summoner ("매치 상세 줘")
                                          ──→ LLM 호출
                                          → 분석 결과 반환
```

유저가 결과를 기다리고 있으므로 전부 REST.

**5. 새 패치가 감지된다**

```
Collector ──Redis Streams──→ Analytics ("새 패치 14.22 나왔어")
           ──Redis Streams──→ Comp ("이전 패치 데이터 아카이브해")
```

패치 전환은 즉시 처리 안 해도 됨 → Redis Streams.

## 통신 요약

| 통신 경로 | 방식 | 시나리오 |
| --- | --- | --- |
| API Gateway → 각 서비스 | REST | 모든 유저 요청 |
| Summoner → Collector | Redis Streams | #2 미수집 소환사 백그라운드 수집 |
| Collector → Analytics | Redis Streams | #3 배치 트리거 |
| Analytics → Collector | REST | #3 배치 중 매치 데이터 조회 |
| Analytics → Comp | REST | #3 집계 결과 전달 |
| Analytics → Summoner | Redis Streams | #3 캐시 무효화 |
| AI Feedback → Comp | REST | #4 덱 통계 조회 |
| AI Feedback → Summoner | REST | #4 매치 상세 조회 |
| Collector → Analytics, Comp | Redis Streams | #5 패치 전환 알림 |

## 이벤트 스트림 (Redis Streams)

| 스트림 | 발행자 | 구독자 | 페이로드 |
| --- | --- | --- | --- |
| `match.batch.ready` | Collector | Analytics | `{ patchVersion, matchCount }` — 배치 트리거 |
| `summoner.collect.request` | Summoner | Collector | `{ puuid, region }` — 미수집 소환사 수집 요청 |
| `stats.updated` | Analytics | Summoner | `{ patchVersion, updatedAt }` — 소환사 캐시 무효화 |
| `patch.detected` | Collector | Analytics, Comp | `{ newVersion, previousVersion }` — 패치 전환 알림 |
