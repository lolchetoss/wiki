## 공통 기술 스택

| 영역 | 선택 |
| --- | --- |
| 프론트엔드 | React |
| 백엔드 | Spring Boot (Java) |
| DB | MySQL |
| 캐시 | Redis |
| 메시지 | Redis Streams |
| 아키텍처 | MSA |
| 인프라 | AWS (EC2 올인원 → 단계적 분리) |

## 서비스별 기술 스택

| 서비스 | 프레임워크 | DB | 캐시 | 외부 연동 | 아키텍처 |
| --- | --- | --- | --- | --- | --- |
| API Gateway | Spring Cloud Gateway | - | - | - | - |
| Comp Service | Spring Boot | MySQL | Redis | - | 클린 아키텍처 |
| Summoner Service | Spring Boot | MySQL | Redis | Riot API | DDD |
| AI Feedback Service | Spring Boot | MySQL | - | Riot API, LLM API | 클린 아키텍처 |
| Auth Service | Spring Boot + Spring Security | MySQL | Redis (세션) | Riot RSO, 결제 API | DDD |
| Collector Service | Spring Boot | MySQL | - | Riot API, S3 | DDD |
| Analytics Service | Spring Boot | MySQL | - | - | 클린 아키텍처 |

### 주요 라이브러리

| 용도 | 라이브러리 |
| --- | --- |
| API Gateway | Spring Cloud Gateway |
| REST 클라이언트 (서비스 간 통신) | Spring WebClient |
| 인증/인가 | Spring Security + JWT |
| Redis 연동 | Spring Data Redis |
| DB 접근 | Spring Data JPA |
| Redis Streams | Spring Data Redis (StreamListener) |
| API 문서 | SpringDoc OpenAPI (Swagger) |
| 테스트 | JUnit 5, Mockito, Testcontainers |
