## MVP 범위

1차 런칭에 포함되는 기능: **추천 덱 조합**, **꿀 덱**, **큐레이션 덱**, **전적 검색**

| 기능 | 핵심 가치 | 데이터 소스 |
| --- | --- | --- |
| 추천 덱 조합 | 현 메타 최적 덱을 통계 기반으로 자동 산출 | 매치 데이터 클러스터링 (Platinum+) |
| 꿀 덱 | 남들이 모르는 저픽률 고승률 조합 자동 발굴 | 매치 데이터 필터링 |
| 큐레이션 덱 | 고수(그마+)의 운영법/노하우 가이드 | 인증 플레이어 직접 작성 |
| 전적 검색 | 내 전적 조회 + 플레이어 태그 + 매치 상세 | Account/Match/League API |

**상세 명세:**
- [[데이터 수집]] — 수집 파이프라인, API Key 전략, 폴링, 에러 처리
- [[덱 시스템]] — 추천 덱, 꿀 덱, 큐레이션 덱, 관리자 수정, 덱 태그
- [[전적 검색]] — 검색 UX, 소환사 프로필, 매치 히스토리, 공유, 플레이어 태그

## 메인 페이지

### 레이아웃 (위에서 아래 순서)

**1. 상단 — 검색 + 패치 정보**
- 소환사 검색바 (메인 CTA, 중앙 배치)
- 현재 패치 버전 + 데이터 기준 정보 (예: "패치 14.21 · 52,341 매치 분석 · 30분 전 갱신")
- 티어 필터 (Platinum+ / Diamond+ / Master+ / Challenger) — 메인에서 선택하면 하위 섹션 모두 반영

**2. 추천 덱 티어리스트 Top 8~10**
- S/A 티어 덱만 표시
- 카드형: 덱 이름, 티어 뱃지, 주력 특성 아이콘, 캐리 챔피언 이미지, 승률, 평균 등수
- 카드 클릭 → 덱 상세 페이지로 이동
- "전체 보기" 버튼 → 덱 페이지 (메타 덱 탭)로 이동

**3. 꿀 덱 하이라이트 Top 3**
- 꿀 점수 높은 순
- 카드형: 덱 이름, 꿀 점수, 승률, 픽률, 발굴 시점
- "전체 보기" 버튼 → 덱 페이지 (꿀 덱만 보기 필터)로 이동

**4. 최신 큐레이션 가이드 Top 3**
- 최근 등록/업데이트 순
- 카드형: 덱 이름, 작성자 닉네임 + 티어 뱃지, 작성일
- "전체 보기" 버튼 → 덱 페이지 (가이드 탭)로 이동

### 갱신

- 메인 페이지 데이터는 통계 배치(1시간)와 동일 주기로 캐싱
- 패치 전환 시 자동으로 새 패치 데이터로 전환 (이전 패치는 드롭다운으로 조회 가능)

## 페이지 구성

| 페이지 | 설명 |
| --- | --- |
| 메인 | 검색바, 추천 덱 Top 8~10, 꿀 덱 Top 3, 최신 큐레이션 가이드 Top 3 |
| 덱 | 메타 덱 + 꿀 덱 + 큐레이션 가이드를 탭으로 통합 |
| 덱 상세 | 덱 구성, 아이템, 증강, 레벨별 가이드, 통계, 팀 코드 |
| 전적 검색 | 소환사 검색, 매치 히스토리, 매치별 상세 |

### 덱 페이지 구조

덱 관련 모든 정보를 하나의 페이지에서 탭으로 제공:

**탭 1: 메타 덱** (기본)
- 상단: 티어 필터 (Platinum+ 등) + 필터/정렬 옵션
- 덱 카드 목록: 티어 뱃지(S/A/B/C) + 덱 태그(Hot/꿀/New 등) 함께 표시
- 꿀 덱은 🍯 태그로 구분, "꿀 덱만 보기" 필터로 모아볼 수 있음
- 기본 정렬: 티어(복합 점수) 높은 순

**탭 2: 가이드**
- 큐레이션 덱 가이드 목록
- 정렬: 최신순 / 인기순
- 필터: 특성, 캐리 챔피언
- 작성자 티어 뱃지 + 해당 덱 승률 표시

## 공통 UX 패턴

### 로딩 상태

- **스켈레톤 UI** 사용 — 콘텐츠 영역의 형태를 유지한 채 회색 placeholder 표시
- 페이지별 스켈레톤:
  - 메인: 검색바는 즉시 표시, 덱 카드 영역 스켈레톤
  - 덱 목록: 카드 스켈레톤
  - 전적 검색: 프로필 카드 + 매치 카드 스켈레톤
  - 매치 상세 (아코디언): 펼침 시 내부 스켈레톤

### 에러 상태

| 상황 | 메시지 | 액션 |
| --- | --- | --- |
| API 장애 / 네트워크 에러 | "문제가 발생했습니다" | 재시도 버튼 |
| 소환사 검색 실패 | "소환사를 찾을 수 없습니다" | 닉네임#태그 형식 안내 |
| Rate Limit 초과 | "요청이 많습니다. 잠시 후 다시 시도해주세요" | 자동 재시도 (Retry-After 기반) |
| 서버 점검 | "TFT 서버 점검 중입니다" | Status API 기반 자동 감지 |

### 빈 상태

| 상황 | 메시지 |
| --- | --- |
| 매치 히스토리 없음 | "최근 매치 기록이 없습니다" |
| 필터 결과 없음 | "조건에 맞는 덱이 없습니다. 필터를 조정해보세요" |
| 패치 초반 데이터 부족 | "새 패치 데이터를 수집 중입니다 (현재 N건)" + 진행률 표시 |
| 꿀 덱 없음 | "현재 패치에서 발견된 꿀 덱이 아직 없습니다" |
| 큐레이션 가이드 없음 | "아직 등록된 가이드가 없습니다" |

### 반응형 웹

웹 기반 반응형으로 데스크톱/태블릿/모바일 모두 대응:

| 뷰포트 | 기준 | 레이아웃 |
| --- | --- | --- |
| 데스크톱 | 1024px 이상 | 전체 레이아웃, 사이드바 필터 |
| 태블릿 | 768~1023px | 2열 그리드, 필터 접기 |
| 모바일 | 767px 이하 | 1열 스택, 필터 모달, 터치 최적화 |

**모바일 주요 대응:**
- 덱 카드: 챔피언 아이콘 크기 축소, 가로 스크롤 없이 표시
- 매치 히스토리: 카드형 유지, 핵심 정보만 (등수, 덱 이름, 챔피언)
- 필터: 하단 모달 시트로 표시
- 매치 상세 8명 보드: 가로 스크롤 허용 또는 축소 뷰
- 툴팁: 호버 대신 탭으로 표시

## API 엔드포인트 (백엔드)

| Method | Path | 설명 |
| --- | --- | --- |
| GET | `/api/comps` | 추천 덱 목록 (필터, 정렬, 페이징) |
| GET | `/api/comps/{id}` | 덱 상세 정보 |
| GET | `/api/comps/honey` | 꿀 덱 목록 |
| GET | `/api/guides` | 큐레이션 덱 가이드 목록 |
| GET | `/api/guides/{id}` | 가이드 상세 |
| POST | `/api/guides` | 가이드 작성 (인증 필요) |
| PUT | `/api/guides/{id}` | 가이드 수정 (작성자만) |
| GET | `/api/summoner/{name}/{tag}` | 소환사 프로필 + 통계 |
| GET | `/api/summoner/{name}/{tag}/matches` | 매치 히스토리 (페이징) |
| GET | `/api/matches/{matchId}` | 매치 상세 (8명 전체) |
| PUT | `/api/admin/comps/{id}` | 관리자 덱 수정 (추천 덱/꿀 덱 보정) |
| DELETE | `/api/admin/comps/{id}/override` | 관리자 수정 리셋 (통계 기반 복원) |
| GET | `/api/meta/summary` | 현재 패치 메타 요약 (메인 페이지용) |
| GET | `/api/patches` | 패치 목록 |
