## 프로젝트 개요

TFT 메타 덱 추천 + AI 분석 리포트를 핵심으로 한 통계 서비스

## 차별점

| 항목     | 기존 사이트     | 롤체토스                          |
| ------ | ---------- | ----------------------------- |
| 광고     | 배너 광고 다수   | 광고 없거나 비침습적 형태만               |
| AI 분석 리포트  | 없음         | 통계 기반 AI 분석 리포트                   |
| 꿀 덱    | 없음         | 저픽률 고승률 조합 자동 발굴              |
| 큐레이션 덱 | 없거나 운영진 작성 | 티어 인증된 고수(그마+)가 직접 작성하는 덱 가이드 |

## 기능 로드맵

### 1차 런칭 — 핵심 가치 + 수익 모델

| 기능 | 핵심 가치 |
| --- | --- |
| 메타 덱 | 메타 덱 자동 산출, 티어 분류, 꿀 덱 자동 발굴 |
| 전적 검색 | 소환사 전적 조회 + AI 분석 리포트 진입점 |
| AI 분석 리포트 (유료) | 통계 비교 + AI 자연어 리포트로 플레이 피드백 |
| 클라이언트 데이터 수집 프로그램 | API 미제공 데이터(라운드별 골드, 리롤 타이밍, 보드 스냅샷 등) 수집 — 오버레이 아님, 순수 수집 목적. AI 피드백의 풍부한 데이터 확보용 |

### 2차 — 콘텐츠 확장

| 기능            | 핵심 가치                         |
| ------------- | ----------------------------- |
| 큐레이션 덱        | 티어 인증된 고수(그마+)가 직접 작성하는 덱 가이드 |
| 증강/아이템/챔피언 통계 | 덱 단위가 아닌 개별 요소 심화 분석          |

### 3차 — 플랫폼 확장

| 기능 | 핵심 가치 |
| --- | --- |
| 인게임 오버레이 | 클라이언트 데이터 수집 프로그램 기반, 실시간 메타 덱/아이템 추천 |
| 커뮤니티 | 덱 토론, 메타 분석 공유 |
| 덱 시뮬레이터 | 챔피언/아이템/시너지 조합 및 예상 성과 확인 |

## 수익 모델

| 구분 | 내용 |
| --- | --- |
| 무료 | 메타 덱, 전적 검색 |
| 무료 체험 | AI 분석 리포트 2회 (회원가입 시 제공) |
| 회당 결제 | 1회 ₩500 / 5회 ₩2,000 / 15회 ₩5,000 |

| 후원 | Buy me a coffee 등 |
| 최소 광고 | 필요 시 비침습적 형태만 |
