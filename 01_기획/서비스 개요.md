## 프로젝트 개요

TFT 메타 덱 추천 + AI 리포트를 핵심으로 한 통계 서비스

## 차별점

| 항목     | 기존 사이트              | 롤체토스                                      |
| ------ | ------------------- | ----------------------------------------- |
| 광고     | 배너 광고 다수            | 광고 없거나 비침습적 형태만                           |
| AI 리포트 | 통계 수치 나열 (직접 해석 필요) | 클라이언트 수집 + Riot API 데이터 → AI 자연어 리포트로 피드백 |
| 메타 덱   | 티어/승률 통계 제공         | 티어 분류 + 꿀 덱(저픽률 고승률) 자동 발굴                |
| 덱 큐레이션 | 없거나 운영진 작성          | 티어 인증된 고수(그마+)가 직접 작성하는 덱 가이드             |

## 기능 로드맵

### 1차

| 기능 | 설명 |
| --- | --- |
| 메타 덱 | 메타 덱 자동 산출, 티어 분류, 꿀 덱 자동 발굴 |
| 전적 검색 | 소환사 전적 조회 + AI 리포트 진입점 |
| AI 리포트 (유료) | 클라이언트 수집 + Riot API 데이터 → AI 자연어 리포트로 플레이 피드백 |
| 클라이언트 | 인게임 데이터 수집 프로그램 (라운드별 골드, 리롤 타이밍, 보드 스냅샷 등). 오버레이 아님, 순수 수집 목적 |

### 2차

| 핵심 기능         | 설명                         |
| ------------- | -------------------------- |
| 커뮤니티          | 덱 큐레이션(그마+가 직접 작성하는 덱 가이드) |
| 증강/아이템/챔피언 통계 | 덱 단위가 아닌 개별 요소 통계          |

### 3차 — 플랫폼 확장

| 기능 | 설명 |
| --- | --- |
| 인게임 오버레이 | 클라이언트 기반 실시간 메타 덱/아이템/증강 통계 조회 |
| 덱 시뮬레이터 | 챔피언/아이템/시너지 조합 및 예상 성과 확인 |
