## 프로젝트 개요

TFT 메타 덱 추천 + AI 게임 피드백을 핵심으로 한 통계 서비스

- metatft급 데이터 깊이 + 롤체지지급 한국어 UX
- 광고 최소화, 깔끔한 UX
- 다국어: 한국어, 영어

## 차별점

| 항목 | 기존 사이트 | 롤체토스 |
| --- | --- | --- |
| 광고 | 배너 광고 다수 | 광고 없거나 비침습적 형태만 |
| AI 분석 | 없음 | 매치 후 AI 게임 피드백 |
| 꿀 덱 | 수동 티어리스트 | 저픽률 고승률 조합 자동 발굴 |
| 한국어 + 데이터 | 한국어 좋으면 데이터 얕고, 데이터 깊으면 한국어 부족 | 둘 다 확보 |

## 기능 로드맵

**1차 런칭**
- 추천 덱 조합 — 패치별 메타 덱 티어리스트, 승률/픽률/등수 통계, 티어별 필터
- 꿀 덱 — 저픽률 고승률 조합 자동 서피싱
- 전적 검색 — 소환사별 매치 히스토리, AI 피드백 진입점
- 큐레이션 덱 — 인증된 고수(그마+)가 직접 작성하는 덱 가이드

**2차 이후**
- 증강/아이템/챔피언 통계 — 독립 페이지로 심화 분석
- AI 게임 피드백 (유료)
- 커뮤니티
- 덱 시뮬레이터

## 무료 / 유료

| 구분 | 기능 |
| --- | --- |
| 무료 | 추천 덱 조합, 꿀 덱, 전적 검색, 기본 통계 |
| 무료 체험 | AI 게임 피드백 2회 (회원가입 시 제공) |
| 유료 | AI 게임 피드백 — 덱 선택 판단, 운영 개선점, 상위권 비교 분석 |

> 전환 유도: 회원가입 시 AI 피드백 2회 무료 체험 제공 → 체험 소진 후 유료 전환 유도. 매치 히스토리에 AI 분석 요약 한 줄을 미리보기로 노출, 전체 분석은 유료 잠금

## AI 게임 피드백

매치 종료 후 데이터 기반 플레이 분석 (유료)

**피드백 예시:**
- "이 덱은 현재 패치 승률 45%입니다. 비슷한 시너지의 이 덱(승률 52%)이 더 나은 선택이었어요"
- "같은 덱 1~2등 유저들은 이 챔피언에 A/B/C 아이템을 줬는데, D를 주셨어요. A로 바꾸면 평균 1.2등수 차이납니다"
- "2차 증강 X를 골랐는데, 이 덱에서는 Y 증강이 승률 12% 높습니다"

## 수익 모델

| 모델 | 설명 |
| --- | --- |
| AI 피드백 회당 결제 | 1회 단건 구매, 묶음 시 할인 (예: 1회 ₩500 / 5회 ₩2,000 / 15회 ₩5,000) |
| 프리즘 티켓 (구독) | 월 ₩9,900 — AI 피드백 무제한 (회당 결제 대비 하루 1회만 써도 이득) |
| 후원 | Buy me a coffee 등 |
| 최소 광고 | 필요 시 비침습적 형태만 |
