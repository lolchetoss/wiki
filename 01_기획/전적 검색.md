## 검색 UX

**검색 입력:**
- 닉네임만 입력 시 → 동일 닉네임의 Riot ID 후보 목록 표시 (태그가 다른 동명이인)
- 닉네임#태그 정확히 입력 시 → 바로 프로필 페이지로 이동
- DB에 있는 유저는 자동완성 후보로 표시 (Riot API에 자동완성 없음)

**동명이인 처리:**
- 닉네임만 검색 시 결과 목록: 닉네임#태그, 티어, 최근 게임 시간 표시
- 유저가 원하는 계정 선택 → 프로필로 이동
- DB에 결과 없으면 "닉네임#태그를 정확히 입력해주세요" 안내 (Account API는 태그 필수)

**최근 검색 / 즐겨찾기:**
- 최근 검색 10명 — 로컬 스토리지 저장 (비로그인 상태에서도 유지)
- 즐겨찾기 — 비로그인: 로컬 스토리지, 로그인: 서버 저장
- 검색창 포커스 시 최근 검색 + 즐겨찾기 목록 드롭다운 표시

**검색 실패:**
- 존재하지 않는 닉네임 → "소환사를 찾을 수 없습니다" + 닉네임#태그 형식 안내
- API 에러 → "잠시 후 다시 시도해주세요"

## 조회 흐름

```
닉네임#태그 입력 → Account API (puuid 조회) → Summoner/League/Match API 호출
닉네임만 입력 → DB 검색 + Account API 조회 → 동명이인 목록 표시
```

## 소환사 프로필

**기본 정보:**
- 닉네임#태그 + 프로필 아이콘
- 랭크 티어/LP + 상위 % + 순위 (예: "Master 309 LP · 상위 0.19% · 3,210위")
- 게임 모드별 랭크 (랭크/초고속/더블업 각각 티어 표시)

**핵심 통계:**
- 승리 횟수 + 승률 (1등 횟수/전체 + 비율)
- Top4 횟수 + Top4 비율 (4등 이내 횟수/전체 + 비율)
- 게임 수 + 평균 등수
- 각 항목별 상위 % 표시 (동일 티어 내 비교, 예: "승률 상위 16.61%")

**그래프/차트:**
- LP/티어 추이 그래프 (날짜별 꺾은선)
- 최근 20게임 등수 격자 (각 게임 등수를 숫자 칸으로 나열, 1~4등 파란색/5~8등 회색)
- 등수 분포 바 차트 (1~8등 각각 몇 번인지 막대)

**최근 N게임 통계:** (기본 20게임, 드롭다운으로 변경 가능)
- 챔피언 통계 Top 5 — 가장 많이 사용한 챔피언 + 게임 수 + 평균 등수
- 시너지 통계 Top 5 — 가장 많이 활성화한 특성 + 게임 수 + 평균 등수
- 증강 통계 Top 5 — 가장 많이 고른 증강 + 게임 수 + 평균 등수
- 선호 덱 Top 3 — 가장 많이 플레이한 조합 (추천 덱 연결)

**플레이 스타일 분석:**
- 플레이어 태그 — "리롤러", "한 우물", "안정러" 등 자동 부여 (아래 참고)
- 플레이 스타일 축 차트 — 유연함↔강제 / 골드 관리 / 탱커↔딜러 / AD↔AP 성향 축
- 코스트 선호도 — 저코 리롤 vs 고코 레벨링 비율 차트

**추가 재미 요소:**
- 같은 티어 평균 대비 — "다이아 평균 등수 4.2인데 당신은 3.8" 비교 카드
- 시즌 최고 기록 — 최고 LP, 최다 연승, 최고 등수 스트릭
- 연승/연패 현황 — "현재 3연속 4등 이내" 또는 "5연패 중" 실시간 표시

## 매치 히스토리

**매치 카드 한 줄에 표시할 정보:**

- 등수 (1~8위, 색상으로 구분 — 1~4등 파란계열, 5~8등 빨간계열)
- 운영한 덱 이름 + 주력 특성 아이콘
- 챔피언 아이콘 나열 (성급 표시 — 별 개수 또는 테두리 색)
- 증강 3개 아이콘
- 게임 모드 (랭크/하이퍼롤/더블업/일반)
- 경과 시간 (예: "2시간 전")

**필터:**

| 필터 | 옵션 |
| --- | --- |
| 게임 모드 | 전체 / 랭크 / 일반 / 하이퍼롤 / 더블업 |
| 기간 | 최근 7일 / 30일 / 시즌 전체 |
| 등수 | 전체 / 1~4등만 / 5~8등만 |

**페이징:** 20 매치씩 무한 스크롤 또는 "더 보기" 버튼

## 매치 상세

매치 카드 클릭 시 아코디언으로 펼침 (페이지 이동 X):

- 8명 전체 참가자 보드 (등수순 정렬)
- 참가자별: 챔피언 구성, 아이템, 특성, 증강, 레벨
- 내 매치와 1등 매치 비교 하이라이트
- 해당 덱의 메타 승률 표시 (추천 덱 연결)
- 참가자 닉네임 클릭 → 해당 소환사 전적 페이지로 이동
- 챔피언/아이템/증강 아이콘 호버 → 툴팁으로 상세 정보

## 공유

**URL 공유:**
- 프로필 URL: `lolchetoss.com/summoner/KR/닉네임/태그`
- 매치 URL: `lolchetoss.com/match/KR_12345678`
- 공유 버튼 → 클립보드 복사 + "복사되었습니다" 토스트

**프로필 카드 이미지 공유:**
- "내 전적 공유" 버튼 → 바로 카드 이미지 생성 (서버사이드 렌더링)
- PNG 다운로드 또는 클립보드 복사
- SNS/디스코드/카카오톡에 바로 붙여넣기 가능한 카드 형태

카드에 포함되는 정보:
- 꼬마 전설이 이미지 + 닉네임#태그
- 티어/LP + 승률 + 게임 수 + 평균 등수
- 최근 20게임 등수 격자
- 플레이어 태그
- 카드 생성일 + 롤체토스 로고/URL 워터마크

## 플레이어 태그

전적 데이터 기반으로 자동 부여하는 재미 요소. 태그에 호버/탭 시 정상적인 설명 툴팁 표시.

| 태그 | 설명 (툴팁) | 판정 기준 |
| --- | --- | --- |
| 🎰 리롤 중독 | 저레벨에서 리롤을 많이 돌려 3성을 노리는 스타일 | 최근 20게임 평균 레벨 7 이하 + 3성 유닛 보유율 높음 |
| 🔋 원기옥 | 초반을 참고 후반에 폭발하는 레벨링 스타일 | 최근 20게임 평균 레벨 8.5 이상 |
| 💀 대깨 | 하나의 덱만 고집하는 장인 스타일 | 최근 20게임 중 동일 덱 50% 이상 |
| 🌊 유연함 | 상황에 따라 매번 다른 덱을 운영하는 적응형 플레이어 | 최근 20게임 중 고유 덱 종류 10개 이상 |
| 👑 메타 추종자 | 현재 메타 강덱 위주로 플레이하는 스타일 | 최근 20게임 중 S/A 티어 덱 80% 이상 |
| 🔧 덱 메이커 | 메타에 없는 덱으로도 높은 등수를 기록하는 창의적 플레이어 | B 티어 이하 덱으로 평균 3등 이내 |
| 🍗 치킨 수집가 | 1등을 자주 차지하는 마무리에 강한 플레이어 | 1등률 20% 이상 |
| 🛡️ 안전 운전 | 꾸준히 상위권에 드는 안정적인 플레이어 | 4등 이내 비율 70% 이상 |
| 🎲 1등 아니면 8등 | 등수 변동이 극단적인 올인 스타일 플레이어 | 최근 20게임 등수 표준편차 2.5 이상 |
