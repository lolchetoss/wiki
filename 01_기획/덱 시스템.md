## 추천 덱 조합

### 덱 분류 로직

1. **캐리 챔피언 식별** — 아이템 3개 보유 챔피언을 메인 캐리로 판별
2. **주력 특성 식별** — `traits[]`에서 `style` 값이 높은(골드/크로매틱) 활성 특성 추출
3. **덱 클러스터링** — 캐리 + 주력 특성 조합으로 1차 분류, 챔피언 구성의 Jaccard similarity로 유사 조합 병합 (유사도 0.6 이상이면 같은 덱으로 병합)
4. **대표 덱 선정** — 클러스터 내 가장 빈도 높은 챔피언 조합을 대표 덱으로
5. **덱 네이밍** — "주력특성 + 캐리 챔피언" 조합으로 자동 생성 (예: "벌꿀 베이가", "선봉대 가렌")

### 덱 티어 산정

복합 점수(Comp Score) 기반으로 S/A/B/C 티어를 자동 부여.

**복합 점수 공식:**

```
score = (9 - 평균등수) / 5 × 0.5
      + 승률 × 0.25
      + 1등률 × 0.15
      + min(픽률 / 10, 1) × 0.1
```

| 요소 | 가중치 | 정규화 방법 | 설명 |
| --- | --- | --- | --- |
| 평균 등수 | 50% | (9 - 평균등수) / 5 → 0~1 | 핵심 지표, 등수가 낮을수록 높은 점수 |
| 승률 (4등 이상) | 25% | 그대로 0~1 | 안정성 지표 |
| 1등률 | 15% | 그대로 0~1 | 폭발력 지표 |
| 픽률 | 10% | min(픽률/10%, 1) → 0~1 | 10% 이상이면 만점, 극소 픽률 패널티 |

**티어 기준:**

| 티어 | 점수 범위 | 의미 |
| --- | --- | --- |
| S | 상위 10% | 현 메타 최강 덱 |
| A | 상위 10~30% | 안정적으로 강한 덱 |
| B | 상위 30~60% | 평균적인 덱 |
| C | 하위 40% | 메타에서 밀리는 덱 |

**최소 조건:**
- 티어 부여 최소 샘플: **50 매치** 이상 (미만이면 "데이터 부족" 표시)
- 패치 초반(1,000 매치 미만): 최소 샘플을 **20 매치**로 완화

### 티어 필터

사용자가 통계 기준 티어를 선택할 수 있음:

| 필터 | 대상 | 용도 |
| --- | --- | --- |
| Platinum+ | 플래티넘 이상 전체 | 기본값, 일반적인 메타 |
| Diamond+ | 다이아몬드 이상 | 중상위권 메타 |
| Master+ | 마스터 이상 | 고수 메타 |
| Challenger | 챌린저만 | 최상위 메타 |

> 티어 필터 변경 시 모든 통계(승률, 픽률, 추천 아이템, 덱 티어 등)가 해당 티어 기준으로 재계산

### 통계 갱신

- **갱신 주기**: 1시간마다 통계 재집계 (배치 작업)
- **실시간 아님**: 수집은 실시간이지만 통계 계산은 배치로 처리 (서버 부하 관리)
- **캐싱**: 티어 필터별 통계를 미리 계산하여 캐싱, API 응답 시 캐시 반환
- **패치 초반**: 데이터 1,000 매치 미만이면 갱신 주기를 30분으로 단축

### 통계 항목

| 항목 | 설명 |
| --- | --- |
| 승률 | 4위 이상 비율 |
| 평균 등수 | 해당 덱 사용 시 평균 placement |
| 픽률 | 전체 매치 대비 해당 덱 등장 비율 |
| 1등률 | 1위 비율 |
| 추천 아이템 | 캐리별 가장 높은 승률의 아이템 조합 (Top 3) |
| 추천 증강 | 해당 덱과 함께 승률이 높은 증강 (1/2/3차별 Top 3) |
| 샘플 수 | 해당 덱의 총 매치 수 (신뢰도 판단용) |

### 필터/정렬

**필터:**

| 필터     | 옵션                                          |
| ------ | ------------------------------------------- |
| 랭크 티어  | Platinum+ / Diamond+ / Master+ / Challenger |
| 덱 티어   | S / A / B / C / 전체                          |
| 주력 특성  | 특성 목록에서 선택                                  |
| 캐리 코스트 | 1코 / 2코 / 3코 / 4코 / 5코                      |

**정렬:**

| 정렬 기준 | 기본 방향 |
| --- | --- |
| 티어 (복합 점수) | 높은 순 (기본값) |
| 승률 | 높은 순 |
| 평균 등수 | 낮은 순 |
| 픽률 | 높은 순 |
| 1등률 | 높은 순 |

### 덱 상세 정보

각 덱 상세 페이지에 표시할 정보:

- 핵심 챔피언 구성 (코스트, 성급 분포)
- 시너지 조합 (활성 특성 목록)
- 아이템 빌드 (챔피언별 추천 아이템 Top 5 + 승률 표시)
- 증강 추천 (1/2/3차 선택 시점별 Top 5, 해당 덱에서의 승률 표시)
- 레벨별 배치 가이드 (6/7/8/9 레벨 시 추천 구성)
- 패치별 승률 추이 그래프
- 관련 큐레이션 가이드 (있는 경우)
- **팀 코드 복사** — 클릭 시 TFT 클라이언트에서 바로 불러올 수 있는 팀 코드를 클립보드에 복사

## 관리자 덱 수정

추천 덱과 꿀 덱은 통계 기반 자동 생성이므로, 관리자가 수동으로 디테일을 보정할 수 있음:

| 수정 가능 항목 | 설명 |
| --- | --- |
| 덱 구성 | 핵심/선택 챔피언 구분, 불필요 챔피언 제거 |
| 덱 이름 | 자동 생성된 이름을 더 직관적인 이름으로 수정 |
| 추천 아이템 | 통계 순위 오버라이드, 챔피언별 아이템 우선순위 조정 |
| 추천 증강 | 통계 순위 오버라이드 |
| 배치 가이드 | 레벨별 추천 구성 직접 편집 |
| 덱 설명 | 한줄 요약, 운영 팁 등 텍스트 추가 |

- 추천 덱, 꿀 덱 모두 동일하게 적용
- 관리자 수정 시 "수동 보정됨" 표시 — 통계와 관리자 의견이 다를 수 있음을 유저에게 알림
- 통계 갱신 시 관리자 수정 내용은 유지 (자동 덮어쓰기 안 함)
- 관리자가 리셋하면 다시 통계 기반으로 복원

## 큐레이션 덱

통계만으로는 운영법, 타이밍, 변형 등을 설명할 수 없음. 인증된 고수가 직접 덱 가이드를 작성하여 보완.

### 등록 자격

- **인증 기준**: 현 시즌 또는 직전 시즌 **그랜드마스터 이상**
- **인증 방법**: Riot 계정 연동 (RSO) → League API로 티어 자동 확인
- **등록자 표시**: 닉네임, 티어 뱃지, 해당 덱 최근 성적

### 가이드 작성 항목

| 항목 | 필수 | 설명 |
| --- | --- | --- |
| 덱 구성 | O | 챔피언 목록 (드래그 앤 드롭으로 배치) |
| 캐리 지정 | O | 메인/서브 캐리 표시 |
| 아이템 빌드 | O | 챔피언별 추천 아이템 + 대체 아이템 |
| 증강 추천 | O | 1/2/3차 증강 우선순위 |
| 레벨별 구성 | - | 6/7/8/9 레벨 시 추천 배치 |
| 운영 가이드 | - | 레벨링 타이밍, 리롤 시점, 이코노미 팁 |
| 변형 덱 | - | 상황별 변형 (메타 카운터, 경합 시 대안) |

> 추천 덱과 마찬가지로 팀 코드 복사 버튼 포함

### 큐레이션 덱 vs 추천 덱

| 구분 | 추천 덱 (자동) | 큐레이션 덱 (수동) |
| --- | --- | --- |
| 생성 방식 | 매치 데이터 클러스터링 | 인증된 플레이어가 직접 작성 |
| 강점 | 객관적 통계, 자동 갱신 | 운영법, 타이밍, 변형 등 노하우 |
| 갱신 | 실시간 (데이터 수집 주기) | 작성자가 수동 업데이트 |
| 신뢰도 | 데이터 기반 | 작성자 티어/성적으로 검증 |

### 품질 관리

- 가이드에 해당 덱의 **실제 통계**(승률, 평균 등수)를 자동 표시 — 통계와 가이드의 교차 검증
- 작성자의 **해당 덱 최근 전적**을 함께 표시 — "이 사람이 실제로 잘하는 덱인가" 확인
- 패치 변경 시 작성자에게 **업데이트 요청 알림** 발송
- 30일 이상 미업데이트 가이드는 "오래된 가이드" 표시

## 꿀 덱

저픽률 고승률 조합을 자동으로 발굴해서 보여주는 기능

### 진입 조건

| 조건 | 기준값 | 설명 |
| --- | --- | --- |
| 픽률 | 5% 이하 | 5% 초과면 이미 알려진 덱 |
| 평균 등수 | 4.0 이하 | 평균 4등 이내 = 확실히 강함 |
| 최소 샘플 | 30 매치 이상 | 노이즈 방지 |
| 패치 초반 완화 | 최소 샘플 15 매치 | 전체 데이터 1,000 매치 미만일 때 |

### 꿀 덱 점수

픽률 대비 성과가 얼마나 좋은지를 수치화:

```
꿀 점수 = (9 - 평균등수) / (픽률% × 100)
```

| 예시 | 평균 등수 | 픽률 | 꿀 점수 | 해석 |
| --- | --- | --- | --- | --- |
| 숨겨진 강덱 | 2.5등 | 1% | 6.5 | 거의 안 쓰이는데 압도적 성적 |
| 일반 꿀 | 3.5등 | 4% | 1.375 | 약한 꿀, 곧 알려질 수 있음 |
| 초희귀 꿀 | 3.0등 | 0.5% | 12.0 | 극소수만 아는 초강덱 |

> 꿀 점수순으로 정렬하여 목록 표시

### 졸업/탈락 조건

꿀 덱은 영구하지 않음 — 조건이 바뀌면 자동으로 상태 전환:

| 상황 | 조건 | 처리 |
| --- | --- | --- |
| 졸업 (메타 진입) | 픽률 5% 초과 | 꿀 덱에서 제거, 추천 덱으로 자동 편입 |
| 탈락 (성과 하락) | 평균 등수 4.0 초과 | 꿀 덱에서 제거 |
| 샘플 부족 | 샘플 수 30 미만으로 감소 | 꿀 덱에서 제거 (패치 전환 시 발생) |

### 표시 정보

- 덱 구성 + 통계 (추천 덱과 동일 포맷)
- **꿀 덱 점수** (높을수록 숨겨진 강덱)
- 발굴 시점 (언제부터 감지되었는지)
- 패치 내 승률 추이
- 현재 픽률 추이 (졸업 임박 여부 확인용)

## 덱 태그

덱 카드에 상태를 한눈에 보여주는 태그 시스템:

| 태그 | 조건 | 색상 |
| --- | --- | --- |
| 🔥 Hot | 최근 24시간 픽률이 직전 대비 50% 이상 급상승 | 빨강 |
| 🍯 꿀 덱 | 꿀 덱 진입 조건 충족 (픽률 5% 이하 + 평균 4등 이내) | 노랑 |
| 🆕 New | 이번 패치에서 새로 감지된 덱 (이전 패치에 없던 클러스터) | 초록 |
| 📈 상승 | 최근 3일 평균 등수가 0.5 이상 개선 | 파랑 |
| 📉 하락 | 최근 3일 평균 등수가 0.5 이상 하락 | 회색 |

> 태그는 중복 부여 가능 (예: Hot + 꿀 덱)
