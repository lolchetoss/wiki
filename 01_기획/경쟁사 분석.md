## 사이트별 분석

### lolchess.gg — OP.GG 계열, 한국 사용자 기반 최대

- **강점**: 한국어 최상, 천상계 관전, 커뮤니티(듀오 찾기), 모바일 앱
- **약점**: 광고 많음, 데이터 분석 깊이 부족

### metatft.com — 프로 플레이어(SpencerTFT) 운영, 데이터 분석 특화

- **강점**: Data Explorer(커스텀 필터 쿼리), 증강 통계 깊이 최상, Overwolf 오버레이
- **약점**: 초기 로딩 느림, 일부 기능 유료

### tactics.tools — 균형 잡힌 통계 사이트

- **강점**: 높은 정보 밀도 + 깔끔한 UI, 치트시트/시너지 그리드, 광고 적음
- **약점**: 오버레이/커뮤니티 없음, 모바일 최적화 부족

### mobalytics.gg — 멀티 게임 플랫폼, 초보자 친화

- **강점**: 쉬운 오버레이, 레벨링 가이드, 가이드 콘텐츠 풍부
- **약점**: 데이터 깊이 부족, 전문가 큐레이션이라 업데이트 느림

## 롤체토스 차별점

경쟁사도 클라이언트/오버레이를 통해 인게임 데이터를 수집한다. 차별점은 데이터 수집 자체가 아니라, 수집된 데이터를 AI로 분석하여 읽기 쉬운 리포트와 구체적인 피드백을 제공하는 것이다.

| 항목 | 경쟁사 | 롤체토스 |
| --- | --- | --- |
| 인게임 데이터 수집 | 오버레이/클라이언트로 수집 | 자체 클라이언트로 수집 |
| 데이터 제공 방식 | 통계 수치 나열 (직접 해석 필요) | AI가 리포트로 작성 + 구체적 피드백 |
| 분석 결과물 | 아이템/증강/구성 통계표 | 등급 평가(S/A/B/C) + 판단 근거 + 개선 제안 |
| 매치 컨텍스트 | 없음 | 로비 분위기 분석 + 컨텍스트 대비 판단 평가 |

핵심: 같은 데이터를 수집하더라도, 유저가 직접 숫자를 해석해야 하는 경쟁사와 달리 롤체토스는 AI가 "왜 졌는지, 어떻게 하면 더 좋았을지"를 자연어로 설명해준다.
