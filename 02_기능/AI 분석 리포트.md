## 개요

매치 종료 후 유저의 플레이를 통계 데이터와 비교 분석하고, AI가 종합 리포트를 생성하는 유료 기능. 데이터 분석은 코드로 정확하게 수행하고, 결과를 AI에 넘겨 자연어 리포트로 변환한다.

## 진입 흐름

전적 검색 → 매치 히스토리 → "AI 분석 리포트" → 잔여 횟수 차감 → 분석 결과 표시

- 잔여 횟수 부족 시 결제 유도
- 매치 히스토리에서 분석 요약 한 줄을 미리보기로 노출하여 전환 유도
- 분석 결과는 저장하여 재열람 가능 (동일 매치 재분석 시 횟수 차감 없음)

## 분석 구조

```
[코드] DB 쿼리 + 통계 비교 → 구조화된 포맷 (예: JSON) 생성 (정확)
    ↓
[Claude API] JSON + Skills(리포트 작성 규칙) → 일관된 자연어 리포트 생성
```

- **데이터 분석**: 코드가 수행. 통계 비교, 최적 아이템/증강 계산, 구성 평가 등을 모두 deterministic하게 처리하여 구조화된 포맷(예: JSON)으로 출력
- **리포트 생성**: Claude API에 JSON + Skills(톤, 포맷, 우선순위 규칙 등)를 넘겨서 일관된 리포트 생성. AI는 숫자를 만들지 않고 이미 계산된 결과를 설명하는 역할만 수행
- **Skills 기반**: 리포트의 톤, 구조, 항목별 표현 방식을 Skills로 정의하여 매번 일관된 품질의 리포트를 보장

## 분석 항목

### 단일 매치 분석

| 항목 | 분석 내용 | 비교 기준 |
| --- | --- | --- |
| 골드 관리 | 남은 골드 과다 시 투자 제안, 골드 효율 평가 | 같은 덱 상위권 |
| 아이템 | 캐리별 최적 아이템 비교, 부품 미완성 감지, 잘못된 챔피언에 아이템 배분 | 같은 덱 상위권 아이템 통계 |
| 증강 | 1/2/3차 증강 선택 평가, 덱 방향과 불일치 감지 (리롤덱에 레벨링 증강 등) | 덱별 증강 순방률 |
| 챔피언 구성 | 핵심 유닛 누락, 시너지 미기여 유닛, 특성 브레이크포인트 놓침 | 같은 덱 상위권 구성 |
| 성급/레벨 | 최종 레벨 비교, 리롤덱 3성 미달, 레벨링덱 레벨 부족 | 같은 덱 상위권 |
| 매치 내 비교 | 1등과 내 보드 diff (아이템, 구성, 증강), 같은 덱 경합 감지 | 같은 매치 참가자 |

### 멀티 매치 분석

| 항목    | 분석 내용                    |
| ----- | ------------------------ |
| 반복 실수 | 최근 N게임에서 반복되는 실수 패턴 감지   |
| 개선 추적 | 지난주 대비 항목별 변화            |
| 증강 습관 | 덱과 무관하게 같은 증강만 고르는 경향 감지 |
| 멘탈 패턴 | 탈락 후 연쇄 하락 경향 감지 (틸트 분석) |

## 분석 불가 영역

라운드별 진행 데이터가 없으므로 다음은 분석 불가:

- 유닛 배치 (위치 데이터 미제공)
- 리롤/레벨업 타이밍
- 이코노미 운영 (라운드별 골드)
- 상점 선택
