## 덱 분류

수집된 매치 데이터에서 주력 특성 + 캐리 챔피언을 기준으로 덱을 자동 분류한다. 덱 이름, 챔피언 구성, 추천 아이템 등은 자동 생성하되, 관리자가 보정할 수 있다.

> 분류 알고리즘의 구체적인 방법은 구현 시 결정.

## 덱 티어

평균 등수, 순방률, 1등 확률, 픽률을 종합한 복합 점수 기반으로 S/A/B/C 티어를 자동 부여한다.

| 티어 | 의미 |
| --- | --- |
| S | 현 메타 최강 덱 |
| A | 안정적으로 강한 덱 |
| B | 평균적인 덱 |
| C | 메타에서 밀리는 덱 |

- 최소 **50 매치** 이상이어야 티어 부여 (미만이면 "데이터 부족" 표시)
- 패치 초반(1,000 매치 미만): 최소 샘플을 **20 매치**로 완화

## 덱 태그

덱 카드에 상태를 한눈에 보여주는 태그. 중복 부여 가능 (예: Hot + 꿀).

| 태그 | 조건 | 색상 |
| --- | --- | --- |
| 🔥 Hot | 최근 24시간 픽률이 직전 대비 50% 이상 급상승 | 빨강 |
| 🍯 꿀 | 픽률 5% 이하 + 평균 등수 4.0 이하 + 30 매치 이상 | 노랑 |
| 🍯🍯 개꿀 | 꿀 조건 충족 + 픽률 2% 이하 + 평균 등수 3.0 이하 | 노랑 (강조) |
| 🆕 New | 이번 패치에서 새로 감지된 덱 | 초록 |
| 📈 상승 | 최근 3일 평균 등수가 0.5 이상 개선 | 파랑 |
| 📉 하락 | 최근 3일 평균 등수가 0.5 이상 하락 | 회색 |

### 꿀 태그 졸업/탈락

| 상황 | 조건 | 처리 |
| --- | --- | --- |
| 졸업 (메타 진입) | 픽률 5% 초과 | 꿀 태그 제거 |
| 탈락 (성과 하락) | 평균 등수 4.0 초과 | 꿀 태그 제거 |
| 샘플 부족 | 샘플 수 30 미만으로 감소 | 꿀 태그 제거 (패치 전환 시 발생) |

> 패치 초반(전체 1,000 매치 미만)에는 최소 샘플을 15 매치로 완화

## 통계

1시간마다 갱신 (패치 초반 30분). 티어 필터 변경 시 모든 통계가 해당 티어 기준으로 재계산된다.

**티어 필터:**

| 필터 | 대상 |
| --- | --- |
| Platinum+ | 플래티넘 이상 전체 (기본값) |
| Diamond+ | 다이아몬드 이상 |
| Master+ | 마스터 이상 |
| Challenger | 챌린저만 |

**통계 항목:**

| 항목 | 설명 |
| --- | --- |
| 순방률 | 4위 이상 비율 |
| 평균 등수 | 해당 덱 사용 시 평균 placement |
| 픽률 | 전체 매치 대비 해당 덱 등장 비율 |
| 1등 확률 | 1위 비율 |
| 추천 아이템 | 캐리별 순방률 높은 아이템 조합 (Top 3) |
| 추천 증강 | 해당 덱에서 순방률 높은 증강 (1/2/3차별 Top 3) |
| 샘플 수 | 해당 덱의 총 매치 수 (신뢰도 판단용) |

## 필터/정렬

**필터:**

| 필터 | 옵션 |
| --- | --- |
| 덱 티어 | S / A / B / C / 전체 |
| 덱 태그 | 꿀 덱만 보기 / Hot / New 등 |
| 주력 특성 | 특성 목록에서 선택 |
| 캐리 코스트 | 1코 / 2코 / 3코 / 4코 / 5코 |

**정렬:**

| 정렬 기준 | 기본 방향 |
| --- | --- |
| 티어 (복합 점수) | 높은 순 (기본값) |
| 순방률 | 높은 순 |
| 평균 등수 | 낮은 순 |
| 픽률 | 높은 순 |
| 1등 확률 | 높은 순 |

## 덱 상세

| 항목 | 설명 |
| --- | --- |
| 챔피언 구성 | 핵심 챔피언, 코스트, 성급 분포 |
| 시너지 | 활성 특성 목록 |
| 아이템 빌드 | 챔피언별 추천 아이템 Top 5 + 순방률 |
| 증강 추천 | 1/2/3차별 Top 5 + 순방률 |
| 레벨별 가이드 | 6/7/8/9 레벨 시 추천 구성 |
| 팀 코드 복사 | TFT 클라이언트에서 바로 불러올 수 있는 팀 코드 |
| 큐레이션 가이드 | 관련 가이드 연결 (2차, 있는 경우) |

## 관리자 보정

메타 덱은 통계 기반 자동 생성이므로, 관리자가 수동으로 보정할 수 있다.

| 항목 | 설명 |
| --- | --- |
| 덱 구성 | 핵심/선택 챔피언 구분, 불필요 챔피언 제거 |
| 덱 이름 | 자동 생성된 이름을 더 직관적인 이름으로 수정 |
| 추천 아이템/증강 | 통계 순위 오버라이드 |
| 배치 가이드 | 레벨별 추천 구성 직접 편집 |
| 덱 설명 | 한줄 요약, 운영 팁 등 텍스트 추가 |

- "수동 보정됨" 표시로 유저에게 알림
- 통계 갱신 시 관리자 수정 내용 유지 (자동 덮어쓰기 안 함)
- 관리자가 리셋하면 통계 기반으로 복원
