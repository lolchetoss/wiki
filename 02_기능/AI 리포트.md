## 개요

매치 종료 후 유저의 플레이를 통계 데이터와 비교 분석하고, AI가 종합 리포트를 생성하는 유료 기능. 클라이언트 프로그램이 수집한 인게임 실시간 데이터와 Riot API의 매치 결과를 결합하여 라운드별 상세 분석까지 제공한다. 데이터 분석은 코드로 정확하게 수행하고, 결과를 AI에 넘겨 자연어 리포트로 변환한다.

## 진입 흐름

전적 검색 → 매치 히스토리 → "AI 리포트" → 잔여 횟수 차감 → 분석 결과 표시

- 잔여 횟수 부족 시 결제 유도
- 매치 히스토리에서 분석 요약 한 줄을 미리보기로 노출하여 전환 유도
- 분석 결과는 저장하여 재열람 가능 (동일 매치 재분석 시 횟수 차감 없음)

## 데이터 소스

| 소스 | 수집 시점 | 제공 데이터 | 필수 여부 |
| --- | --- | --- | --- |
| 클라이언트 프로그램 | 인게임 실시간 | 라운드별 골드/XP, 리롤 타이밍, 아이템 조합 순서, 보드 스냅샷, 데미지 로그, 증강 선택지 | 필수 |
| Riot API | 매치 종료 후 | 최종 보드 상태, 8명 참가자 정보, 등수, 특성, 아이템, 증강 | 필수 |
| 메타 덱 통계 | 사전 집계 | 덱별 순방률, 아이템/증강 통계, 상위권 구성 | 필수 |

> 클라이언트 데이터가 없는 매치는 분석 대상에서 제외된다. 클라이언트 프로그램을 실행한 상태에서 플레이한 매치만 AI 리포트를 생성할 수 있다.

## 분석 구조

```
[클라이언트 프로그램] 인게임 실시간 수집 → DB 저장
                                            ↓
[매치 종료] Riot API 매치 결과 수집 ──────→ 클라이언트 데이터 + API 데이터 매칭
                                            ↓
[코드] 통합 데이터 + 메타 덱 통계 비교 → 구조화된 포맷 (JSON)
                                            ↓
[Claude API] JSON + Skills → 자연어 리포트 생성
```

- **데이터 통합**: 클라이언트 수집 데이터(라운드별 과정)와 Riot API 데이터(최종 결과)를 매치 ID로 매칭하여 통합
- **데이터 분석**: 코드가 수행. 라운드별 운영 평가, 통계 비교, 최적 아이템/증강 계산 등을 deterministic하게 처리하여 JSON 출력
- **리포트 생성**: Claude API에 JSON + Skills를 넘겨 일관된 리포트 생성. AI는 컨텍스트 분석과 자연어 설명을 수행
- **Skills 기반**: 리포트의 톤, 구조, 항목별 표현 방식을 Skills로 정의하여 일관된 품질 보장

## 등급 체계

종합 등급 + 카테고리별 등급 4개를 S/A/B/C로 산출한다.

| 카테고리 | 평가 기준 |
| --- | --- |
| 운영 | 레벨업 타이밍, 골드 관리, 이자 유지율, 리롤 효율 |
| 덱 선택 | 덱 티어, 로비 내 경쟁자 수, 핵심 유닛 경합도 |
| 아이템 | 캐리/서포터별 아이템 최적도, 완성 타이밍 |
| 증강 | 1/2/3차 증강 각각의 덱 시너지 적합도 |

- 종합 등급: 4개 카테고리의 가중 평균
- 등급 기준은 동일 덱 + 동일 티어 구간의 상위 N% 기반으로 산출

## 매치 컨텍스트 분석

개별 플레이 평가에 앞서 판 전체의 분위기를 분석한다.

- **로비 태그**: 판의 특성을 태그로 요약 (예: 느린 템포, 고이자 로비, 공격적 로비)
- **8명 평균 행동 패턴**: 평균 레벨업 타이밍, 리롤 시점, 이자 유지율
- **플레이어 판단 평가**: 로비 컨텍스트 대비 플레이어 선택의 적절성

매치 컨텍스트는 "왜 이 선택이 좋았는지/나빴는지"를 설명하는 근거로 사용된다.

| 증강 | 1/2/3차 증강 선택 평가, 미선택 증강 분석, 덱 방향 일치도 | 클라이언트 + API | 덱별 증강 순방률 |

| 전투 분석 | 라운드별 데미지 로그, 포지셔닝 효율 | 클라이언트 | 매치 내 상위권 |
| 매치 내 비교 | 1등과의 diff (아이템, 구성, 증강), 같은 덱 경합 감지 | API | 같은 매치 참가자 |

## 결제 모델

| 구분 | 내용 |
| --- | --- |
| 회당 결제 | 건당 구매 (가격 미정) |
| 구독 (프리즘 티켓) | 월정액 무제한 (가격 미정) |
| 무료 체험 | 회원가입 시 N회 제공 |

- 매치 분석 요청 시 1회 차감
- 분석 결과는 캐싱되어 재조회 시 차감 없음
- 마이페이지에서 잔여 횟수 확인 + 충전 가능

## Riot API 단독 분석의 한계

Riot API만으로는 최종 보드 상태만 확인 가능하다. 클라이언트 프로그램으로 수집하는 인게임 데이터가 있어야 다음 분석이 가능해진다:

| 분석 항목 | Riot API만 | 클라이언트 + API |
| --- | --- | --- |
| 라운드별 골드/레벨 변화 | 불가 | 가능 |
| 리롤/레벨업 타이밍 | 불가 | 가능 |
| 아이템 조합 순서 | 불가 | 가능 |
| 스테이지별 보드 변화 | 불가 | 가능 |
| 전투 라운드별 데미지 | 불가 | 가능 |
| 증강 미선택지 분석 | 불가 | 가능 |
| 최종 보드 구성/아이템/증강 | 가능 | 가능 |

경쟁사도 클라이언트/오버레이로 유사한 인게임 데이터를 수집하지만, 수치를 나열하는 수준에 그친다. 롤체토스의 차별점은 이 데이터를 AI로 분석하여 등급 평가 + 판단 근거 + 개선 제안을 자연어 리포트로 제공하는 것이다.
