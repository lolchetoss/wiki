## 검색

- 닉네임#태그 정확히 입력 시 → 바로 프로필로 이동
- 닉네임만 입력 시 → DB에서 동명이인 후보 목록 표시
- DB에 있는 유저는 자동완성 후보로 표시
- 최근 검색 10명 + 즐겨찾기 기능

## 소환사 프로필

**기본 정보:**
- 닉네임#태그 + 프로필 아이콘
- 랭크 티어/LP + 상위 % + 순위
- 게임 모드별 랭크 (랭크/초고속/더블업)

**핵심 통계:**
- 1등 횟수 + 1등 확률
- 순방 횟수 + 순방률 (4등 이내)
- 게임 수 + 평균 등수
- 각 항목별 동일 티어 내 상위 % 비교

**추이/분포:**
- LP/티어 추이
- 최근 20게임 등수 분포
- 등수별 횟수 분포 (1~8등)

**최근 N게임 통계:** (기본 20게임, 변경 가능)
- 챔피언 Top 5
- 시너지 Top 5
- 증강 Top 5
- 선호 덱 Top 3 (메타 덱 연결)

**플레이 스타일 분석:**
- 플레이어 태그 자동 부여 (아래 참고)
- 성향 분석 — 유연함↔강제 / 탱커↔딜러 / AD↔AP
- 코스트 선호도 — 저코 리롤 vs 고코 레벨링

**재미 요소:**
- 같은 티어 평균 대비 비교
- 시즌 최고 기록 (최고 LP, 최다 연승 등)
- 연승/연패 현황

## 매치 히스토리

매치별 표시 정보: 등수, 덱 이름, 챔피언 구성, 증강, 게임 모드, 경과 시간

**필터:**

| 필터 | 옵션 |
| --- | --- |
| 게임 모드 | 전체 / 랭크 / 일반 / 하이퍼롤 / 더블업 |
| 기간 | 최근 7일 / 30일 / 시즌 전체 |
| 등수 | 전체 / 1~4등만 / 5~8등만 |

## 매치 상세

매치 선택 시 상세 정보 표시:

- 8명 전체 참가자 보드 (등수순)
- 참가자별: 챔피언 구성, 아이템, 특성, 증강, 레벨
- 내 매치와 1등 매치 비교
- 해당 덱의 메타 순방률 표시
- 참가자 닉네임 → 해당 소환사 전적으로 이동

## 플레이어 태그

전적 데이터 기반으로 자동 부여하는 재미 요소.

| 태그 | 설명 | 판정 기준 |
| --- | --- | --- |
| 🎰 리롤 중독 | 저레벨 리롤로 3성을 노리는 스타일 | 최근 20게임 평균 레벨 7 이하 + 3성 유닛 보유율 높음 |
| 🔋 후반 전문가 | 초반을 참고 후반에 폭발하는 레벨링 스타일 | 최근 20게임 평균 레벨 8.5 이상 |
| 💀 대깨 | 하나의 덱만 고집하는 장인 스타일 | 최근 20게임 중 동일 덱 50% 이상 |
| 🌊 유연함 | 매번 다른 덱을 운영하는 적응형 | 최근 20게임 중 고유 덱 종류 10개 이상 |
| 👑 메타 추종자 | 메타 강덱 위주로 플레이 | 최근 20게임 중 S/A 티어 덱 80% 이상 |
| 🔧 덱 메이커 | 비메타 덱으로도 높은 등수 | 최근 20게임 B 티어 이하 덱으로 평균 3등 이내 |
| 🍗 치킨 수집가 | 1등을 자주 차지하는 플레이어 | 최근 20게임 1등 확률 20% 이상 |
| 🛡️ 안전 운전 | 꾸준히 상위권에 드는 안정적 플레이어 | 최근 20게임 순방률 70% 이상 |
| 🎲 1등 아니면 8등 | 등수 변동이 극단적인 올인 스타일 | 최근 20게임 등수 표준편차 2.5 이상 |
| 🔪 살인광 | 다른 플레이어를 많이 처치하는 플레이어 | 최근 20게임 처치 수가 상위 10% |
| 💥 딜 머신 | 높은 피해량을 기록하는 플레이어 | 최근 20게임 총 피해량이 상위 10% |
| 💎 고벨류 | 고코스트 유닛 위주로 보드를 구성 | 최근 20게임 보드 총 코스트가 상위 10% |
| 🌈 프리즘 컬렉터 | 크로매틱 티어 특성을 자주 활성화 | 최근 20게임 크로매틱 특성 보유 비율 높음 |
| 💰 금고지기 | 골드를 잘 안 쓰는 스타일 | 최근 20게임 평균 남은 골드 높음 |
| 🔥 순방 행진 | 연속으로 순방 중 | 최근 N게임 연속 4등 이내 |
| 🥶 하락세 | 연속으로 탈락 중 | 최근 N게임 연속 5등 이하 |
| 🧊 강철 멘탈 | 탈락 후에도 흔들림 없는 플레이어 | 탈락 직후 게임 성적이 평소와 차이 없음 |
| 💔 유리 멘탈 | 한번 무너지면 연쇄 탈락하는 경향 | 탈락 직후 게임 성적이 평소보다 크게 하락 |
| 🚀 흐름 타는 자 | 순방 후 연속 순방하는 경향 | 순방 직후 게임 성적이 평소보다 상승 |
| 🎯 XX 애호가 | 특정 덱을 자주 사용하는 플레이어 | 최근 20게임 중 특정 덱 5회 이상 사용 (덱 이름으로 동적 생성) |
